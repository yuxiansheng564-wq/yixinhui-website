## 📝 配置文件

### 文件位置
```
/workspace/projects/videos-final.json
```

### 配置格式

```json
{
  "位置": "首页-本周AI-tips-1",
  "标题": "视频标题",
  "视频链接": "B站嵌入代码或本地视频路径"
}
```

---

## 🔗 视频链接格式

### 1. 本地视频（< 50MB）
```json
"视频链接": "/videos/xxx.mp4"
```

### 2. B站嵌入代码（推荐）✅
```json
"视频链接": "//player.bilibili.com/player.html?isOutside=true&aid=xxx&bvid=xxx&cid=xxx&p=1"
```

### 3. B站视频页面链接（需要我解析）
```json
"视频链接": "https://www.bilibili.com/video/BV1xx411c7mD"
```

## 📖 获取B站嵌入代码

### 步骤：

1. **打开B站视频页面**
2. **点击"分享" → "嵌入"**
3. **复制嵌入代码中的 `src` 属性**

**示例：**

原始嵌入代码：
```html
<iframe src="//player.bilibili.com/player.html?isOutside=true&aid=123&bvid=BV1xx&cid=123&p=1"></iframe>
```

复制的内容：
```
//player.bilibili.com/player.html?isOutside=true&aid=123&bvid=BV1xx&cid=123&p=1
```

## ⚠️ 注意事项

1. **位置标识不能重复**
   - 每个 `"位置"` 必须唯一

2. **B站嵌入代码格式**
   - 去掉 `https:`（例如 `//player.bilibili.com/...`）
   - 不要包含 `<iframe>` 标签

3. **本地视频**
   - 文件必须先上传到 `/public/videos/` 目录
   - 路径以 `/videos/` 开头

4. **JSON格式**
   - 逗号不能缺失
   - 引号必须使用双引号
