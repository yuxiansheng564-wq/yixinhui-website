# Coze智能体欢迎消息和引导问题配置指南

## 📋 说明

意心之镜的欢迎消息和引导问题现在会从Coze智能体（ID: 7481860697149407270）自动获取。

---

## 🎯 工作原理

### 当前实现：

1. **用户打开意心之镜**
   - 自动调用Coze API发送空查询
   - 获取智能体的欢迎消息和引导问题
   - 在聊天界面显示

2. **欢迎消息格式**
   - 智能体返回的助手消息
   - 作为"你好！我是意心之镜..."显示

3. **引导问题提取**
   - 从欢迎消息中提取
   - 支持格式：
     ```
     建议问题：
     - 什么是ASOP？
     - 如何用AI提升销售？
     ```

4. **降级方案**
   - 如果API调用失败
   - 使用默认欢迎消息和引导问题

---

## 🔧 在Coze智能体中配置欢迎消息

### 步骤 1：登录Coze平台

1. 访问 https://www.coze.cn
2. 登录您的账号
3. 找到您的智能体（ID: 7481860697149407270）

---

### 步骤 2：配置人设与回复逻辑

1. **点击智能体**
2. **进入"人设与回复逻辑"**
3. **在"开场白"或"欢迎消息"中配置**

---

### 步骤 3：编写欢迎消息（推荐格式）

**格式1：包含引导问题** ✅ 推荐

```
你好！我是意心之镜，您的AI商业提效专家。我可以帮您：

建议问题：
- 什么是ASOP？
- 如何用AI提升销售？
- 企业知识库怎么搭建？
- 意心智造的服务流程是什么？
```

**格式2：纯欢迎消息**

```
你好！我是意心之镜，您的AI商业提效专家。有什么可以帮您的吗？
```

---

### 步骤 4：配置智能体的自动回复

**方法A：使用开场白功能（如果有）**

1. 在智能体配置中找到"开场白"或"欢迎语"
2. 直接粘贴欢迎消息

**方法B：使用触发词**

1. 在"回复逻辑"中添加触发词
2. 触发词：`*` 或 `开场`
3. 回复内容：欢迎消息

**方法C：使用变量**

1. 在"人设"中描述欢迎行为
2. 示例：
   ```
   你是意心之镜，AI商业提效专家。
   当用户第一次对话时，主动介绍自己并提供引导问题。
   引导问题包括：什么是ASOP、如何用AI提升销售等。
   ```

## 🎨 引导问题格式要求

### 支持的格式：

**格式1：列表格式（推荐）**
```
建议问题：
- 问题1
- 问题2
- 问题3
```

**格式2：编号格式**
```
建议问题：
1. 问题1
2. 问题2
3. 问题3
```

**格式3：无格式**
```
欢迎消息内容

什么是ASOP？
如何用AI提升销售？
```

---

## ⚠️ 注意事项

1. **引导问题数量**
   - 建议：2-4个
   - 太多会占用屏幕空间

2. **问题长度**
   - 建议：不超过15个字
   - 太长会换行显示

3. **欢迎消息长度**
   - 建议：不超过100字
   - 太长会影响用户体验

4. **特殊字符**
   - 引导问题支持中文标点
   - 避免使用特殊符号

## 🆘 故障排查

### 问题1：欢迎消息不显示

**可能原因：**
- Coze API调用失败
- 智能体未配置欢迎消息

**解决方案：**
- 检查浏览器控制台错误
- 在Coze智能体中配置欢迎消息
- 检查API Token是否正确

---

### 问题2：引导问题未提取

**可能原因：**
- 格式不符合要求
- 欢迎消息中没有引导问题

**解决方案：**
- 使用推荐格式（建议问题：）
- 确保引导问题在欢迎消息中

---

### 问题3：显示默认欢迎消息

**可能原因：**
- API调用失败
- 智能体返回空内容

**解决方案：**
- 检查网络连接
- 检查Coze API Token
- 确认智能体ID正确
